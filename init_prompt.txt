You are a senior software architect and quantitative developer.

Design and implement a production-grade Automated Trading Bot platform integrating:

* eToro API (or simulated wrapper if API is limited)
* Interactive Brokers (IBKR API)

## 🎯 Objectives

Build a scalable, maintainable, and secure system that:

* Executes automated trading strategies
* Supports multiple brokers (eToro + IBKR abstraction)

* Ensures high reliability and fault tolerance

## 📊 Core Features

### 1. Broker Integration Layer

* Create a Broker interface:

  * placeOrder()
  * cancelOrder()
  * getPositions()
  * getMarketData()
* Implement:

  * IBKRBrokerService
  * EtoroBrokerService (mock or unofficial wrapper)

### 2. Trading Engine

* Strategy interface:

  * generateSignal()
* Implement:

  * Moving Average Strategy
  * RSI Strategy
* Risk management:

  * stop-loss
  * take-profit
  * max drawdown protection

### 3. Market Data Streaming

* Use WebSockets or IBKR streaming API
* Store latest ticks in Redis

### 4. Order Management System (OMS)

* Track:

  * orders
  * executions
  * positions
* Ensure idempotency

### 5. Scheduler

* Run strategies periodically
* Use Quartz or Spring Scheduler
